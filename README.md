# Beacon Supporter
 
Beacon Supporter is a program used for recording digital continuity indormation in no time when we are shooting on set. It was created based on Pyside and Python.

 How to use it :
- Double click the EXE file and good to go
- Click generate button to generate information template and use clear button to clean up the information
- For adjusting the original python file, use requirements.txt to install the module we need
